# Quantitative & Financial Examples

Archive of supporting examples originally published alongside articles on mathematics, quantitative analytics, and financial modelling at `quantandfinancial.com`.

## Context

This repository contains educational material and code samples connected to earlier writing about quantitative finance. It reflects an interest in financial modelling and analytical problem solving, but it is not maintained as a production application or a current portfolio flagship.

## Portfolio Status

Historical public archive. For a modern finance/data engineering portfolio, prefer projects that include reproducible datasets, tests, documented methodology, measurable results, and an accessible demonstration.
